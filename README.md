# Одностраничник для книги

Все анимационные движения реализованны на CSS. Сайт адаптивный для просмотра с телефонов.

**Стек технологий:**
- HTML5 и CSS3
- Препроцессор Sass(SCSS)
- Vue Cli
- JavaScript(отправка сообщения с помошью ajax)
